# NurseIQ Study Coach

A nursing study web app built to help students practice NCLEX-style questions, flashcards, and track study progress.

## Features
- Topic-based quizzes with rationales
- Flashcards for concept review
- Progress tracking (questions, scores, mastery)
- Study session tracker (time logged per subject)

## Tech Stack
- HTML
- CSS
- JavaScript (vanilla)
- LocalStorage for persistence

## Purpose
Built as a personal project to improve nursing exam preparation and learning efficiency.
